General usage:
$ ta <flags> <optional-path>
== Flags ==
-i - view with devicons
-w - view with color
-a - show hidden files
-e - show which .sh files are executable with '*' suffix
== Subcommands ==
tree <level> - view as tree hierarchy
    example usage:
    $ ta tree 3 -i -a ~/Downloads/
help - view available flags, subcommands etc.
    example usage:
    $ ta help
